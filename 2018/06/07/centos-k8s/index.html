安装教程

  • 关闭 CentOS 自带的防火墙服务：
  • 安装 etcd 和 Kubernetes软件(会自动安装Docker):
  • 修改配置文件, 解决在执行命令 kubectl get rc 时会出现 No API token found for service account “default”, retry after the token is automatically create and added to service account 的问题 。
